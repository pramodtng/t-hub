  • HRMS

    HRMS (Human Resource Management System) is a system developed to ease the task of HRs. The HRMS has a feature to apply leave, encash leave, apply for Samsung phones, to view holiday list. Moreover, an attendance of an individual employee is reflected in the HRMS.

  • IP-Phone Directory

    Is an in-house project developed to display the required details of an employee to contact. The IP-phone directory has an email address, extension number and personal phone number of an employee.

  • Selfcare Web

    It is a self-care application for Tashicell mobile subscribers which allows them to view their KYI details, balance and usage details, subscribe and view data plans, manage network settings (like 2g, 3g, 4g and 5g), etc.,

  • Employee Feedback System

    Employee feedback system is developed with a goal to get feedback from an employee. The HR sets a question for an employee which is being notified to an employee. Later the employee answer the question along with emoji.

  • Alert Management System

    It is an application to be used by the different departments of Tashicell to set up alerts and notifications for various events such as vehicle license renewals, lease renewals, subscriptions renewals, etc., It sends notifications via SMS

  • Recruitment Management System

    A system which helps both the HR and job applicant to track the status of an applicant. Once the applicant apply for job, an application number will be sent to an applicant. The system accepts the documents which are being set for as criteria for an individual requirements

  • Office Suite

    Office suite is a collection of application to automate Tashicell internal processes, which are Performance Management System, Document Management System, Alert Management System, Employee Feedback System & Office Recruitment

  • MIS Report Dashboard

    MIS report dashboard is a system developed to display the daily recharge income of data and talk time from various FIs. It also displays the monthly revenue report and interconnect report.

  • Reconciliation System

    It is a system developed to reconcile the daily recharge transactions of My TashiCell app, FIs and eTeeru.

  • AND Ticketing and Reporting System

    Ticketing and Reporting system is develop for AND which helps the technician located at different regions to raise a ticket if any problems occur. The supervisors will be notified if any of the technician raises the ticket.

  • Inventory

    It is a system developed to track an item reaching to office and to its end user. It involves IT support and Inventory Officer the most. It has a feature which helps supervisor to distribute items. After receiving, a confirmation by an employee, transfer of items within regions, surrender of items.
